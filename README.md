# 🌧️ Kerala Weather Info Tool

A **lightweight, fast, and extensible weather utility** for fetching real-time weather data across Kerala.
Built with a clean architecture and minimal dependencies.

---

## ✨ Features

* 🌡️ Real-time weather data
* 📍 Kerala-wide location support
* ⚡ Fast execution
* 🧩 Easy to extend
* 📊 Clean output
