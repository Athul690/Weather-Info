# 🌧️ Kerala Weather Info Tool

A **lightweight, fast, and extensible weather utility** for fetching real-time weather data across Kerala.
Built with a clean architecture and minimal dependencies, this project is designed for reliability today and scalability tomorrow.

---

## ✨ Key Features

* 🌡️ **Real-time Weather Data** — Accurate current conditions
* 📍 **Multi-location Support** — Works across Kerala districts & cities
* ⚡ **High Performance** — Fast execution with minimal overhead
* 🧩 **Modular Design** — Easy to extend and integrate
* 📊 **Clean Output** — Structured and readable CLI results

## ⚙️ How It Works

This tool follows a simple and efficient pipeline:

* ▶️ Execute the script
* 📍 Accept or define a location
* 🌐 Send a request to a weather API
* 📥 Parse the response data
* 📊 Display formatted weather details

The architecture is intentionally minimal, making it easy to:

* Extend functionality (forecast, alerts)
* Integrate with web/mobile interfaces
* Automate periodic weather checks
* Use as a base for larger applications
