## 🚀 Overview

**Kerala Weather Info Tool** is a lightweight command-line application that delivers **real-time weather data** across multiple locations in Kerala.

Built with a clean and modular approach, this project focuses on:

* ⚡ Speed and efficiency
* 🧩 Simplicity and maintainability
* 🔧 Flexibility for future expansion

Whether you're a developer exploring APIs or building utilities, this tool provides a **solid and professional base**.

## ⚙️ How It Works

The application follows a simple yet efficient workflow:

1. ▶️ Execute the script
2. 📍 Input or define a location
3. 🌐 Send request to weather API
4. 📥 Receive structured JSON response
5. 📊 Parse and display formatted output

This design ensures:

* Minimal processing overhead
* Fast API interaction
* Easy debugging and upgrades
